    Surprised at how easy/straightforward it was to get this blog up and running using Publii. I've owned this domain for a little while now and I remember looking at some similar (closed-source) tools in the past, but wasn't too impressed. Think that I was browsing Github's 'Trending Repositories' page when I came across Publii's repo, and was pretty impressed by their site.

    With regard to my current setup, I'm using a Github Pages backend (link to the Github repo) redirected to from a custom domain name (thedeep3nd.io) - Publii handles most/all of the boilerplate JS/CSS/HTML code necessary to link everything up. I looked into some other options - namely, Jekyll and Octopress - when deciding on a framework for the site, but Publii had a couple things that really appealed to me and eventually won me over:

    • great desktop client for Mac
    • accessible/concise documentation
    • ability to build/preview local changes before deploying (committing) to the Github repo
    • an active and growing open-source community
    • free to use!

    I'm sure that it's not the most feature-rich or customizable CMS tool out there, but it seems to have pretty much everything that I want: simplicity, ease of use, and (hopefully) dependability.
